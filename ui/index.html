Data Visualizations

Insights from the thyroid ultrasound dataset

Feature 1 — Class Distribution

Interpretation: The dataset contains ~2879 thyroid ultrasound images split between benign and malignant classes. Class weights are applied during training to prevent the model from biasing toward the majority class.

Feature 2 — Model Performance Radar

Interpretation: Recall is prioritised over Precision in medical imaging — a missed malignant case (false negative) carries far greater clinical risk than a false alarm (false positive).

Feature 3 — Training vs Validation AUC (v2 vs v3)

Interpretation: v2 showed a large train/val gap indicating overfitting. v3 applied AdamW, stronger dropout, and fewer unfrozen layers — closing the gap while maintaining strong validation AUC ~0.80.

Feature 4 — Prediction Confidence Distribution

Interpretation: A well-calibrated model pushes predictions toward the extremes (0 or 1), not the middle. Separation between benign and malignant distributions indicates strong discriminative power.
